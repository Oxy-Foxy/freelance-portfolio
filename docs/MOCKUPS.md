## What we drop from mockups (design rules)

- Card / inset hero media
- Floating badges, promo chips, stat strips on hero
- Cluttered first viewport (address, schedule, “this week”)
- Inter / Roboto / system as display fonts
- Purple gradients, cream+#terracotta serif clichés
- Multi-layer shadows, glow, emoji decoration
